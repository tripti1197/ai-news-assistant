# 🧠 AI News Assistant

An AI-powered web application that generates **summary, key points, and headlines** from news content.

Built using **Angular + Node.js**, this project demonstrates full-stack integration and real-world architecture.

---

## 🚀 Features

- ✨ Generate concise **summary**
- 📌 Extract **key points**
- 📰 Generate **headlines**
- ⚡ Fast backend processing
- 🔄 Loading state handling
- 📋 Copy-to-clipboard support

---

## 🏗️ Tech Stack

**Frontend**
- Angular (Standalone Components)
- TypeScript

**Backend**
- Node.js
- Express.js
